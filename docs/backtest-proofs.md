# backtest-proofs (archived)

This project has been archived. The work is preserved at
[`archive/position-ledger/`](https://github.com/eigenq-xyz/quant-proofs/tree/main/archive/position-ledger)
in the repository.

**What it was:** 26 Lean 4 theorems proving correctness of a position ledger —
mark-to-market identity, cash flow tracking, self-financing constraints — with a
delta-hedging simulation running on top as a demo.

**Why archived:** Pivoting to build a proper backtester from first principles,
starting with the Discrete FTAP (Harrison-Pliska 1981) as the theoretical foundation.
See [`ftap-proofs`](ftap-proofs) for the current work.
